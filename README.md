# Coding Cheat Sheets

A collection of personal cheat sheets created while learning web development.  
These notes are written as I learn new concepts so I can quickly reference them later.

## Purpose

The goal of this repository is to:

- Keep track of useful shortcuts and syntax
- Reinforce concepts while learning
- Build a quick reference I can return to while coding
- Document things I discover during my learning journey

These cheat sheets are not meant to be complete documentation.  
They are simple reminders of things I’ve learned.

## Current Cheat Sheets

- HTML / Emmet – shortcuts and syntax for writing HTML faster
- More cheat sheets will be added as I learn new topics.

## Tools

Most of these notes are used while working in:

- Visual Studio Code
- HTML
- CSS
- JavaScript (later)

## Learning Path

These notes are created while working through:

- The Odin Project

## How I Use These Notes

When I learn something new I:

1. Test it in my editor
2. Add it to the cheat sheet
3. Use it again in practice projects

This helps reinforce what I’ve learned and makes it easier to remember later.
